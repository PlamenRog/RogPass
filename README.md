<p style="text-align:center"><em>A somewhat safe password manager written in c</em></p>


Dependencies:
 - GCC (tested on version 11.4 with c18 and c2x but should work on older ones as well)
 - OpenSSL (>=3.0.0)

Compile it with:
```make
make install
```
